    • vs2015 安装前端 combres 插件

    最近对一个使用webform, asp.net的老项目动手术。 使用combres对前端的css, js 进行压缩合成为单个css.

    安装简单

    1. 直接用 NuGet 安装, 安装完后，会在web.config里自动添加section等配置信息。

    2. 删除　App_code/ App_Start 下的文件Combres.cs， 并在Global.asax文件中添加代码

    3. 在App_Data下会自动生成配置文件combres.xml 等文件， 修改

    4. 在输入的地方只要简单的两行即可

    5. 安装完成。
